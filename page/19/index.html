中秋月饼（枣泥馅）

发表于 2017-01-30 | 分类于 中秋月饼 | 0 comments



原料
枣泥馅：
干红枣一包（400g），水4.5杯，黄砂糖1杯，植物油4.5（爱厨用的是熟花生油），淀粉1.5匙。
模子： 125g 月饼模
