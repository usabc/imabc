蜜汁排骨

发表于 2017-02-28 | 分类于 烧烤 BBQ ， 肉类食物 ， 猪肉 | 0 comments



原料
猪排骨，白糖，精盐，白酒，香油，老抽，柱候酱，芝麻酱，五香粉，茴香粉。

制作方法

（1）将猪排骨清理干净，放开水锅中烫一下捞出，晾凉，白糖熬成浆备用。

（2）把凉好的排骨放在容器内，再将其它调料放入，拌匀腌6小时，使调味品渗透入味。

（3）用叉烧铁环将派规穿上·，再把穿好的排骨放入烤炉，用中火烤40分钟，出炉后淋上糖浆即成。

特点
甘香浓化而带有蜜汁味。
